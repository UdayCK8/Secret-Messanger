🔐 Secret Message Encoder/Decoder

This project is a simple web-based application that allows users to encode and decode secret messages using a custom key-based shifting algorithm. The tool is built using HTML, CSS, and JavaScript, making it fully interactive and runnable directly in a web browser without any backend.

The encoder works by shifting each character in the message using a numeric key provided by the user. The decoder reverses the process using the same key to retrieve the original message. This creates a basic form of custom encryption, useful for learning how data encoding works.

The application features a clean and user-friendly interface where users can:

Enter a message
Choose encode or decode mode
Provide a secret key
View the processed output instantly

This project is ideal for beginners to understand:

Character encoding and ASCII values
Basic encryption logic
Frontend development using JavaScript
